# KC - Bienes Raíces 🏠

Proyecto de sitio web para la inmobiliaria **Ventas y Cambalaches**.

## 🚀 Descripción
Este es el repositorio oficial del sitio web, enfocado en la gestión inmobiliaria y ventas.

## 🛠️ Tecnologías utilizadas
* HTML5
* CSS3 (Diseño de Navbar con contraste de lujo)
* Git & GitHub

## 🎨 Diseño
* **Colores principales:** Azul Marino y Dorado Champagne.
* **Logo:** Isotipo vectorial (KC) con tipografía personalizada.
